## 📁 Projects

### 1. Police Recorded Crime Analysis (2012-2024)
**Dataset:** UK Home Office — England & Wales Crime Data  
**Tools:** Python & Excel  
**What I did:**
- Merged 12 years of crime data (254,429 rows) from multiple sheets
- Cleaned data — removed duplicates and extra columns
- Built 3 Pivot Table analyses with charts
- Wrote professional insights report

**Key Findings:**
- Crime increased 66% from 2012 to 2023
- Metropolitan Police recorded the most crimes (8.1M offences)
- Theft offences were the most common crime type (33.3%)
