# Project 1 — Police Recorded Crime Analysis (2012-2024)

## Overview
Analysis of UK police recorded crime data across England and Wales 
from 2012/13 to 2023/24.

## Dataset
- Source: UK Home Office
- Records: 254,429 rows × 8 columns
- Period: 12 financial years

## Tools Used
- Python (Pandas, Matplotlib)
- Microsoft Excel (Power Query, Pivot Tables, Charts)

## Key Findings
- Crime increased 66% from 2012 to 2023
- Metropolitan Police recorded most crimes (8.1M offences)
- Theft offences most common crime type (33.3%)
- Fraud offences growing rapidly (14% of all crime)
